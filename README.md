# Proyecto de venta de entradas

## Descripción 
Proyecto académico del curso de Desarrollo de Aplicaciones Web (DAW).

Sitio web que permite a los usuarios consultar información sobre un concierto y completar un formulario de compra de entradas, mostrando posteriormente una página de confirmación.

Desarrollado con HTML, CSS y JavaScript.

---

## Contenido del proyecto
**index.html** → página principal con informacón del concierto y formulario de compra de entradas.
**confirmacion.html** → página donde se muestran los datos ingresados.
**Style.css** → estilos para todo el sitio, siguiendo un diseño limpio y responsivo.
**formulario.js** → función 'enviar datos()' que toma la información recopilada del formulario y redirige a 'confirmacion.html' con dichos parámetos.
**confirmacion.js** → función que muestra los datos enviados en 'confirmacion.html'.
**img** → imagen utilizada en el proyecto.

---

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript

---

## Como ejecutar
1. Abrir el Index y rellenar el formulario con la información requerida.
2. Aceptar los términos de uso y hacer click en **Continuar**
3. Se abrira 'confirmacion.html' mostrando la información ingresada como confirmación de pago.
